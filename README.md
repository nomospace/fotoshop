# Fotoshop 🎨

> 在线图片处理工具 - 基于 HTML5 Canvas

## 功能特性

### 📊 基础调整
- 亮度调节
- 对比度调节
- 饱和度调节
- 色相旋转
- 模糊效果

### 🎭 滤镜效果
- 黑白
- 复古（复古棕褐色）
- 反色
- 怀旧
- 冷色
- 暖色
- 戏剧
- 褪色

### ✨ 特效工具
- 马赛克
- 锐化
- 浮雕
- 边缘检测
- 噪点
- 像素化

### 🔄 变换
- 左右旋转 90°
- 水平/垂直翻转

### 📜 操作历史
- 撤销/重做
- 重置为原图

## 使用方法

1. 点击上传区域或拖拽图片
2. 使用左侧工具栏调整图片
3. 点击「下载图片」保存

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3
